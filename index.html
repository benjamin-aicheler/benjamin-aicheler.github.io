Homelab & Security

vaultwarden-export-rclone

A robust utility designed to secure self-hosted password vaults. It automatically exports your Vaultwarden vault database and configuration, encrypts the output, and syncs it to offsite cloud storage endpoints using **rclone**.

  • Zero-Downtime Snapshots: Safely copy the SQLite database using transactional backups to avoid corruption.
  • Multi-Destination Support: Syncs to Google Drive, AWS S3, OneDrive, WebDAV, or SFTP through rclone.
  • Automation Ready: Tailored for cron scheduling, includes retention cleanup, and emits diagnostic logs.
